# 💊 QRMedix – Medicine Tracking System

## 📌 Description
A QR-based system to scan medicines, store details, track expiry, and manage location.

## 🚀 Features
- QR Code Scanning
- Medicine Storage (SQLite)
- Search Functionality
- Expiry Alerts
- Dashboard Analytics
- Login & Signup

## 🛠️ Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS, JavaScript
